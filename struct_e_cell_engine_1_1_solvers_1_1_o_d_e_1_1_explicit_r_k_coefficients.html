Computes the sum of the bs_{i}*k_{i} coefficients for i in [1, s].

This is valid for a unique equation informed by _eqIdx. In particuler, it allows select the appropriate k_{i} in ks.

Parameters
_eqIdxThe index of the equation for which we compute a coefficient k.

const unsigned short  _stage 
)
noexcept

Computes the sum of the a_{stage,j}*k_{j} coefficients for j in [1, s-1].

The minimal value of _stage is 2, the maximal value is stages. This is valid for a unique equation informed by _eqIdx. In particuler, it allows select the appropriate k_{j} in ks.

Parameters
_eqIdxThe index of the equation for which we compute a coefficient k.
_stageThe stage of the coefficient k we are computing.

float* ECellEngine::Solvers::ODE::ExplicitRKCoefficients::as = nullptr

@biref The coefficients a_{i,j} of all stages and all ks.

The array is of size NBAS_PER_STAGE[stages] with the coefficient for stage 2, then the 2 coefficient of stage 3, then the 3 coefficients of stage 4, then the 6 coefficients of stage 5, etc.

It is of size stages * polynomial order. The definition of this array mimmics a matrix of stages rows and polynomial order columns.
