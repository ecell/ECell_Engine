Detailed Description

template<typename TargetType, typename ThresholdType>
class ECellEngine::Core::Trigger< TargetType, ThresholdType >

Class that can be used to watch a condition and trigger events when this condition is fulfilled.

The target and threshold must implement all binary operators used in the comparison. Namely: >, >=, ==, !=, <, <=.

Template Parameters
TargetTypeType of the object being watched. Must be a pointer.
ThresholdTypeType of the object to which the target is being compared.
