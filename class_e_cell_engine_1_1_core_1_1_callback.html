Detailed Description

template<typename ... Args>
class ECellEngine::Core::Callback< Args >

A callback system inpired by the C# one.

The object making the subscription is responsible for keeping the returned shared_ptr alive. It is used as a token to identify the subscription when we want to unsubscribe. Also, if the object is destroyed, the callback will be eventually removed from the list of subscribers.
